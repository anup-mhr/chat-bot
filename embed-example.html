Dynamic Chatbot Embedding Example

This page demonstrates the dynamic chatbot embedding that automatically resizes based on its state. The chatbot starts as a small circular button and expands when opened.

How it works:

  • Closed State: 70x70px circular button in bottom-right corner
  • Open State: 400x600px chat window with rounded corners
  • Dynamic Resizing: Smooth transitions between states
  • No Interference: Doesn't block website content

Test Website Interaction

Click these buttons to verify the website remains fully interactive:

<!-- Simple one-line embedding --> <script src="http://localhost:3000/embed.js" data-auto-init="true" data-url="http://localhost:3000"> </script>

The chatbot will appear as a floating button in the bottom-right corner. When you click it, it will smoothly expand to show the full chat interface. When closed, it shrinks back to the button size.
